# Cultural Advice Bias Visualization

An interactive visualization tool that analyzes and displays epistemic erasure and cultural bias in therapy recommendations across different cultural backgrounds (Filipino, Indian, and Nigerian).

## Project Structure

```
cultural-therapy-bias/
├── epistemic_violence_visualization.html   # Main visualization interface
├── start_server_fixs.py                    # Local server to run the visualization
│
├── Main Analysis Scripts
│   ├── filipino_main_therapy_bias.py
│   ├── indian_main_therapy_bias.py
│   └── nigerian_main_therapy_bias.py
│
├── URL Analyzers
│   ├── filipino_url_analyzer.py
│   ├── Indian_url_analyzer.py
│   └── nigerian_url_analyzer.py
│
├── Therapy Bias Demos
│   ├── filipino_therapy_bias_demo.py
│   ├── Indian_therapy_bias_demo.py
│   └── nigerian_therapy_bias_demo.py
│
├── Assets
│   ├── pinkspace.png
│   ├── orangespace.png
│   ├── greenspace.png
│   └── greyspace.png
│
└── Results (generated by analysis)
    ├── filipino_therapy_bias_results.json
    ├── indian_therapy_bias_results.json
    └── nigerian_therapy_bias_results.json
```

## Prerequisites

- Python 3.7+
- OpenAI API key (or relevant AI service API key)

## Setup Instructions

### 1. Clone the Repository

```bash
git clone https://github.com/YOUR_USERNAME/cultural-advice-bias.git
cd cultural-advice-bias
```

### 2. Install Dependencies

```bash
pip install -r requirements.txt
```

### 3. Configure API Keys

Create a `.env` file in the project root:

```bash
cp .env.example .env
```

Edit `.env` and add your API key:

```
OPENAI_API_KEY=your_api_key_here
```

**Important:** Never commit your `.env` file or API keys to GitHub!

### 4. Run the Analysis (Optional)

If you want to generate new results:

```bash
# Run analysis for each cultural group
python filipino_main_therapy_bias.py
python indian_main_therapy_bias.py
python nigerian_main_therapy_bias.py
```

This will generate/update the JSON result files.

### 5. Start the Visualization

```bash
python start_server_fixs.py
```

Then open your browser to the URL shown (typically `http://localhost:8000`)

## Features

- Interactive visualization of epistemic erasure in therapy recommendations
- Comparative analysis across Filipino, Indian, and Nigerian cultural contexts
- Visual representation using custom icons and data-driven graphics
- JSON-based result storage for easy data sharing and analysis

## Usage

1. Start the server using `start_server_fixs.py`
2. Open the visualization in your web browser
3. Explore the interactive elements to understand cultural biases in therapy recommendations

## Data Files

The project uses JSON files to store analysis results:
- `filipino_therapy_bias_results.json`
- `indian_therapy_bias_results.json`
- `nigerian_therapy_bias_results.json`

These files contain structured data about bias patterns, recommendations, and epistemic erasure indicators.
